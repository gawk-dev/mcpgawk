# mcpgawk — brand spec (Nativerse / gawk.dev family)

mcpgawk is a **gawk.dev-family** open tool under **Nativerse Ventures**. Its identity inherits the Nativerse
system — it does **not** invent its own. (The legacy "gawk black/teal" is superseded by the royal-blue convergence.)

## Palette (locked to Nativerse guidelines)
| Token | Hex | Use |
|---|---|---|
| Surface (dark) | `#16160F` | product / social / OG / terminal background — the family is dark-mode |
| Accent — royal blue | `#2832C2` | the **only** brand accent |
| Accent on dark | `#7B83F0` | same hue, lightened for legibility on `#16160F` |
| Status — live/open | `#36C28B` | semantic green only (the `●` status dot; "clean / scanned / open") |
| Ink | near-white on dark; near-black `#16160F` on light | text |
| Rules | hairline grey | dividers only |

One accent. Semantic green for status only. Hairline rules. Generous whitespace. Editorial restraint.

## Type
- **Wordmark:** lowercase `mcpgawk` in **monospace** — on-brand because the family reserves mono for
  data/proof, and mcpgawk *is* a CLI/measurement tool. Differentiates the product tier from the `nativerse`
  Sentient wordmark.
- **Display/editorial:** serif (Charter/Hoefler) — for any long-form/report surface.
- **Data/proof:** monospace (the scan output already is).

## Mark
- The signature glyph is the tool's own **`●` status dot** (green `#36C28B` = clean/live, royal blue for brand
  contexts) sitting before the monospace wordmark — it *is* what the CLI prints. Simple, ownable, scannable as
  a favicon.
- Parent endorsement: pair with the Nativerse interlocking-N monogram + "a nativerse tool" where lockup space allows.
- Favicon = the `●` dot (royal blue on dark, or green for "live").

never adjectives.
