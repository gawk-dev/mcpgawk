The MCP token tax

You pay it on every prompt — before you type a word.

Connect an MCP server and it loads every tool definition into your model — used or not, on every request. We measured the ones developers actually use.

The two most-installed dev-tool MCPs: 35,600 tokens at connect. Roughly a quarter of your context window — gone before your first prompt.

The surprise: it isn't how many tools. It's how heavy. The heaviest MCP packs 20 tools into 23,085 tokens. Cloudflare fits 23 into 3,570 — 6.5× lighter, same job.

Measured locally with mcpgawk. Heavy servers anonymised; the lean ones named with credit. Nothing uploaded.

And heavy isn't just expensive. In controlled tests, once the tool list gets long the model picks the wrong tool more often — one study: 13.6% right, versus 43.1% when the list is trimmed.

The fix already exists.

Cloudflare: 23 tools in 3,570 tokens, destructive ones labelled. Exa: ships two, lets you add more. Lean is a choice.
